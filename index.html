Arquitectura

Cómo funciona

Tres capas simples que se comunican en tu red local.

1

Servidor Express local

MacroDev levanta un servidor en el puerto :4000 que expone la API y sirve el panel web. Corre silenciosamente en segundo plano gracias a Electron.

2

Extensión de VS Code

Una extensión ligera escucha en el puerto :4001 y traduce las órdenes del servidor en terminales integradas dentro de VS Code.

3

Panel en el móvil

Escaneas el QR con tu teléfono y accedes al panel desde el navegador. Todo en la misma red WiFi, sin latencia perceptible.

4

Detección de proyectos

Al iniciar, MacroDev escanea tu carpeta raíz buscando package.json en cada subdirectorio y genera los controles automáticamente.

Instalación

En marcha en 3 pasos

Requisitos: Node.js v18+ y VS Code en Windows 10/11.

Terminal — Windows PowerShell / CMD
1

Clona o descarga el repositorio en tu carpeta de proyectos:

git clone https://github.com/Zireuz/macroweb.git PORTAFOLIO/macroweb
2

Instala las dependencias (incluye Electron):

cd PORTAFOLIO/macroweb && npm install
3

Instala la extensión de VS Code:

code --install-extension macrodev-terminal-1.0.0.vsix

Reinicia VS Code. Luego haz doble clic en iniciar.vbs para arrancar.
